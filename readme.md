# 概要
これまで影響を受けた本を紹介するアプリです。

# 開発環境・使用言語
MAMP  
HTML・Sass(SCSS)・Laravel・Vue.js・MySQL

# 機能
・会員関連(新規登録・ログイン・ログアウト・アカウント削除・プロフィール画像登録・プロフィール編集・パスワードリマインダー)  
・投稿関連(新規投稿・画像登録・削除・編集)  
・いいねボタン(axiosを使った非同期通信)  
・フラッシュメッセージ表示
